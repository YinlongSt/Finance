## Net Present Value Rule (NPV)
FV = PV(1+r) ; r is interest rate

C stands for cashflow
PV = C/(1+r)
```
NPV = C0 + C1/(1+r)
C0 : negative of the cost of the investment (-)
C1 : Pay off (+)
```

- Accept projects if NPV > 0
- Reject if NPV < 0
